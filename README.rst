====================
Poker Hand Strengths
====================

A collection of poker hands, covering 7 hand types, in the comma-separated values (CSV) format.

Contents:

- Badugi
- Eight-or-better
- Kuhn poker
- Ace-to-five (regular)
- Short-deck hold 'em (6+ hold 'em)
- Deuce-to-seven (standard)
- Deuce-to-seven badugi (standard badugi)
